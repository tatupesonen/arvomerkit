Opi Puolustusvoimien arvomerkit.  
Sovellus toimii Progressive Web Appina, joten sitä voi käyttää vaikka metsän siimeksestä jonne ei netti kuulu.  
PWA tulee avata kerran jotta se lataa kaikki sotilasarvojen tunnukset talteen.

Sovellus on saatavilla osoitteessa: **[https://tatupesonen.github.io/arvomerkit](https://tatupesonen.github.io/arvomerkit)**
